# Интернет-магазин

Это проект интернет-магазина, разработанный с использованием Django и Celery.

## Описание

Данный проект представляет собой веб-приложение для интернет-магазина, позволяющее пользователям просматривать товары, добавлять их в корзину и оформлять заказы.

## Основные функции

- Каталог товаров
- Корзина покупок
- Оформление заказов
- Система управления заказами для администраторов
- Интеграция с платежной системой Stripe

## Технологии

Проект использует следующие основные технологии:

- Django
- Celery
- Redis
- Postgresql
- Stripe API

## Зависимости

Список основных зависимостей проекта находится в файле `requirements.txt`.
